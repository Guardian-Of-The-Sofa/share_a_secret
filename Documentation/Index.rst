=============================================================
Share a secret
=============================================================

:Version:
   1.0.0

:Language:
   en

:Authors:
   Jens Pausewang

:Email:
   jens@hauptsache.net

:License:
   This extension documentation is published under the
   `CC BY-NC-SA 4.0 <https://creativecommons.org/licenses/by-nc-sa/4.0/>`__ (Creative Commons)
   license

This Extension allows you to easily share an encrypted message via a link.
Only people knowing the password to the secret message can view it.

If you need to share messages which should be encrypted and/or are cumbersome to deliver, this
extension is for you.
